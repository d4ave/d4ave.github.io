CHESS



1. Whites will start the game and make their first move.
2. Each player will have 60 seconds to make a move.
And if he doesn't make his move in that time, the opponent will be declared as the winner.
3. The whole game should be completed in 45 minutes.
After this time, the game will be declared drawn.
4. The game can be paused at any point of time, but the game board will not be displayed until the game is resumed.
